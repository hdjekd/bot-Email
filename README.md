
# MR Email Bot 🤖

بوت تليجرام متخصص في إنشاء بريد مؤقت (Temporary Email) مع نظام متكامل.

## المميزات ✨

- 📧 إنشاء بريد مؤقت من 27 دومين مختلف
- 👥 نظام تسجيل المستخدمين بأرقام الهواتف
- 👑 لوحة تحكم خاصة للمطور
- 📊 نظام مراقبة متقدم مع واجهة ويب
- 🔒 إخفاء تام للمصادر
- 🌐 دعم اللغتين العربية والإنجليزية

## الأوامر 📌

| الأمر | الوصف |
|-------|-------|
| /start | بدء البوت |
| /phone | تسجيل رقم الهاتف |
| /generate | إنشاء بريد مؤقت جديد |
| /id | عرض قائمة بريدك |
| /set | إضافة بريد مخصص |
| /fetch | جلب الرسائل |
| /view | عرض آخر رسالة |
| /delete | حذف بريد |
| /domains | عرض الدومينات |
| /language | تغيير اللغة |

## أوامر المطور (مخفية) 👑

| الأمر | الوصف |
|-------|-------|
| /Mydeveloper | لوحة تحكم المطور |
| /monitor | رابط مراقبة البوت |
